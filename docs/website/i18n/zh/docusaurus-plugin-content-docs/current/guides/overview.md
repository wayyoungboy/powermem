## 生态系统集成 {#ecosystem-integrations}

一方集成，为现有的 AI 客户端和 IDE 提供持久记忆——所有客户端都指向相同的后端，无需每个客户端的模式重写。

- **[生态系统集成](../integrations/overview.md)** - 所有 AI 客户端和 IDE 集成的概览
- **[Claude Code](../integrations/claude_code.md)** - `memory-powermem` 插件：通过钩子实现静默 HTTP 模式捕获，可选 MCP 模式用于聊天工具
