# PowerMem Codex Stop

Resolve `PLUGIN_ROOT` from `PLUGIN_ROOT`, `CODEX_PLUGIN_ROOT`, or the installed plugin path under `~/.codex/plugins`, then run:

```bash
sh "$PLUGIN_ROOT/scripts/stop.sh"
```

Report whether a managed PowerMem backend process was stopped. Do not remove memories or config unless the user explicitly asks.
